# Logo Customization Guide

## How to Customize Your Logo

The Kili Smile theme now includes advanced logo customization options available through the WordPress Customizer.

### Accessing Logo Controls

1. **Login to WordPress Admin**
2. **Go to Appearance > Customize**
3. **Click on "Site Identity"**
4. **Find the Logo Settings section**

### Available Controls

#### Logo Size
- **Range**: 20px to 100px
- **Default**: 45px
- **Use**: Adjust the width and height of your logo
- **Responsive**: Automatically scales on mobile devices (75% of desktop size)

#### Logo Border
- **Type**: Checkbox (On/Off)
- **Default**: Off
- **Use**: Adds a colored border around your logo
- **Style**: Creates a circular border effect

#### Border Radius
- **Range**: 0% to 50%
- **Default**: 0%
- **Use**: Controls how rounded the logo border appears
- **Note**: Only visible when "Logo Border" is enabled

#### Border Color
- **Type**: Color Picker
- **Default**: Primary Green (#4CAF50)
- **Use**: Choose the color of the logo border
- **Note**: Only visible when "Logo Border" is enabled

### Features

- **Live Preview**: All changes appear instantly in the Customizer preview
- **Responsive Design**: Logo automatically adjusts for mobile devices
- **Professional Integration**: Uses WordPress best practices
- **Performance Optimized**: CSS is generated dynamically and cached

### Tips

1. **Logo Size**: For best results, keep logo size between 35-60px for desktop
2. **Border Effects**: Use borders for circular logos or to add emphasis
3. **Color Coordination**: Choose border colors that match your brand
4. **Mobile Considerations**: Remember that logos will be smaller on mobile devices

### Technical Details

- Logo settings are saved in WordPress database
- Custom CSS is generated dynamically
- All changes are theme-specific
- Settings persist through theme updates (when properly implemented)

### Troubleshooting

If logo customization isn't working:
1. Clear any caching plugins
2. Refresh the Customizer
3. Ensure you have proper WordPress permissions
4. Check that the theme is active
